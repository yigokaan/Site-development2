Register Features Bucket Module The module that you can define your data structure. This will give you the flexibility
that you need.
 Template Data Models To make a fast start, we set some bucket structure templates. You can
choose a template and update it as your wishes.
 Restful API & Realtime Data Transfer Access all your bucket data in a few milliseconds. Spica allows you to reach
your data realtime as well.
 File Management Store your videos, images, sounds or any kind of videos.
 Multilingual Content Management To support as many as language, you don’t need to write any code. Also you
can define which contents are translatable.
 Frontend Composer The huge module that gives you to compose your user interface without
writing code unless you use custom components. If you decide to develop
unique UI, just push your components to NPM and import into Frontend
Composer.
 Custom Functions The module which gives you to writing your own logic easily. Also you can
set subscribers and webhooks in harmony with Bucket Module.
 Building Module You can take composer builds as mobile application or web application.
Web applications will be PWA supported built-in.
 Auto-Publishing Schedule your posts and let them become visible in the future.
You don't need to be on your computer.
 Policy Management The IAM system which gives you full control on your instance. You can set
policies, identities, and roles that you need.
 History You can see all changes related to a specific entry so you can revert your entry
or track changes easily.
 Assetstore You can find ready-to-use functions or components on Assetstore.
 Single Sign-On Single Sign-On based on SAML 2.0 will give you to manage your users easily.

to changing game. We provide solution focused support 365 days.
